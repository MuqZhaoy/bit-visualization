人格的色彩

每种人格都带着独特的色彩。有些人热情如火、有些人冷艳如冰。本研究正是用可视化地手段，将MBTI理论中所涉及到的人格多样性用直观的可视化方式呈现出来，人生原本多彩。

如今普及度最高的人格分类法

学术背景

MBTI（Myers-Briggs Type Indicator）是基于卡尔·荣格心理类型理论的个性测试工具，由凯瑟琳·布里格斯和伊莎贝尔·布里格斯·迈尔斯在20世纪中期开发。MBTI通过评估外向/内向、感觉/直觉、思考/情感、判断/知觉四个维度，帮助人们了解自己的性格类型。

社交属性

MBTI已经成为了部分年轻群体交友的重要参考标签之一，MBTI正在成为陌生人之间的常见的破冰话题。许多年轻人将自己的MBTI类型挂上社交首页，以此彰显着自己的社交身份，并将MBTI类型作为交友的判准：INFP和ENFJ是“灵魂伴侣”， INTJ和ESFP能成为很好的合作伙伴……

基于BART模型的模型构建

本模型是基于BART模型，采用seq to seq的方式来解决Aspect-based sentiment analysis (ABSA)问题。基于方面的情感分析（ABSA）旨在预测给定文本的情感元素组成的元组。ABSA研究的主要线包括四个情感元素：方面术语（a）、方面类别（c）、意见术语（o）和情感极性（s）。给定一个示例句子，“I love the sushi badly!”（我非常喜欢寿司！），相应的元素分别是“寿司”、“食品质量”、“喜欢”和“积极”。

外向 E 与 内向 I

E 和 I 代表着迈尔斯-布里格斯类型指标（MBTI）中的外向（Extraversion）和内向（Introversion）。外向型人格者通常喜欢社交活动，他们倾向于在与他人的互动中获得能量，并在社交场合表现出开放和积极的态度。相对地，内向型人格者则更倾向于独处或与小团体的深入交流，他们从内部世界的思考和反省中汲取能量。全球范围内，外向与内向的人群比例分布各异，这反映了不同文化背景下对社交行为的不同期待和鼓励。下图直观显示了世界各地外向型人格者的占比情况，通过这种可视化，我们可以更好地理解全球文化差异如何塑造个体的性格和社交偏好。

感觉 S 与 直觉 N

S 和 N 在迈尔斯-布里格斯类型指标（MBTI）中分别代表感觉（Sensing）和直觉（Intuition）。感觉型的人格者注重现实和细节，依赖于通过感官获取的具体信息，而直觉型的人则更倾向于关注未来的可能性和抽象概念，信赖直觉和想象力。这两种认知风格在全球范围内的分布反映了不同文化对实际经验和创新思维的偏好。下图提供了一个直观的视图，展示了世界各地感觉型人格者的占比情况，通过颜色的深浅可以看出各地区对于感觉型人格的普及程度，揭示了全球文化差异如何影响个体的感知方式和决策过程。

情感 F 与 思维 T

F 和 T 在迈尔斯-布里格斯类型指标（MBTI）中分别代表情感（Feeling）和思维（Thinking）。情感型人格者在决策时倾向于强调个人情绪和价值观，注重人际关系的和谐及他人的感受。相对地，思维型人格者则更依赖于逻辑分析和客观事实，追求效率和原则的一致性。全球不同地区对这两种决策方式的偏好反映了文化背景的影响。下图提供了一个全球视角，展示了情感型人群的占比，通过颜色的深浅，我们可以看到各地区对情感或逻辑取向的倾向，进而理解文化如何塑造人们的决策和交际方式。

知觉 P 与 判断 J

P 和 J 在迈尔斯-布里格斯类型指标（MBTI）中代表知觉（Perceiving）和判断（Judging）。判断型的人喜欢有序和计划性地处理事务，更倾向于事先组织和确定，追求明确的结论和条理。相比之下，知觉型的人更加灵活和开放，他们喜欢在保持选项开放的状态下应对生活，更能适应不断变化的环境。这种性格差异在全球范围内的分布揭示了不同文化对结构和自由流动性的不同重视程度。例如，在高度重视效率和规则的社会中，判断型个体可能占多数；而在强调灵活性和 spontaneity 的文化中，知觉型个体更为常见。下图直观地显示了全世界范围内知觉型人群的占比，揭示了全球各地对于这两种处理方式的偏好，从而深入理解文化如何影响人们的行为和决策方式。

社交媒体上不同人格的存在感

虽然不同人格类型的绝对数量在全球范围内差异可能不大，但在互联网上，不同人格类型的表达和参与度却表现出明显的差异。为了探究这一现象，我们选取了一个具体的时间截面来进行分析，专注于统计这一期间内不同人格类型在社交媒体上的活动量。通过这种方法，我们能够量化并比较各人格类型的“存在感”，即他们在数字空间中的活跃程度和影响力。

INFP: 小蝴蝶

INFP（内向、直觉、情感、知觉）是一种由迈尔斯-布里格斯类型指标（MBTI）定义的人格类型，通常被称为“调停者”。这种人格类型以其理想主义、深厚的情感和丰富的想象力著称。INFP 通常具有强烈的理想主义和愿景，关注道德和伦理问题，追求公正和平等。作为内向型人格，INFP 更喜欢独处以反思和充电，并通过写作、艺术或其他创造性活动表达自己。INFP 对自己的情感非常敏感，并且能够深刻理解他人的感受，成为优秀的倾听者和同情心丰富的朋友。他们富有创造力和洞察力，喜欢探索新的想法和可能性。INFP 通常灵活且适应性强，喜欢开放的、非结构化的环境。他们在帮助和支持他人方面非常出色，但也可能过于理想化和情绪敏感。INFP 在需要创意和个人表达的职业中表现出色，例如作家、艺术家、心理咨询师、社会工作者和教师。
